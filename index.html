Hello World.

Jade+Stylus+Grunt = amazing.

Node.JS provides a powerful platform for programming the generation of static HTML files. This site is the result of an example build system.

Source Code Birds-eye

Features

Current:

  • Markup
    • HTML5 Boilerplate inspired template
    • Inlined yepnope script
    • Jade Template Inheritance, Includes, and Locals
  • Styles
    • Fluidity Stylus framework
    • Roots.cx Stylus mixins
    • Solarized colors adapted from LESS
    • Can include vendor css as needed
    • Mobile-Friendly with responsive breakpoint
  • Scripts
    • Runs clientside scripts through Uglify linting & minification
    • Scripts are injected with yepnope
  • Tests
    • Build process testing with Mocha, Should, and Sinon
    • Automated on GitHub push with Travis-CI
  • Misc
    • Grunt Task Chains for scripts, styles, and markup
    • Grunt Watch for kicking off tasks
    • Grunt Connect for serving static site over HTTP
    • Grunt Pygmentize for Generating a Birds-eye Source Code View

Planned:

  • JavaScript dependency management
  • More CoffeeScript
  • More Links
  • More Tests, probably using Jasmine
